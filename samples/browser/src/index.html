Hosted verification

  1. Open your tenant workspace in FaultLens.
  2. Open the target project that matches the project API key you used.
  3. Open Events and search for the smoke ID or release shown above.
  4. Open the event detail to confirm the environment and release values match this sample submission.
  5. Confirm diagnostics context includes the requested URL, browser user agent, demo user ID, and sample tags.
